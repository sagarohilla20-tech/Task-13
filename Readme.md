# Task 13: Odd or Even Checker

## Overview
Welcome to **Task 13**! This is a simple beginner-friendly JavaScript project that checks whether given numbers are **odd** or **even**.

## How to Run
1. Open `index.html` in your browser
2. Press `F12` → Go to **Console** tab
3. You'll see output like:
   - "The given number 1 is Odd."
   - "The given number 2 is Even."
   etc.

## What You’ll Learn
- Variables
- Modulo operator (`%`)
- `if...else` conditions
- `console.log()`

The project includes clear explanations, project structure, learning points, and improvement ideas for beginners.
